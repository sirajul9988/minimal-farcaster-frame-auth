# Minimal Farcaster Frame Auth

A streamlined repository for developers building on the Farcaster protocol. This project utilizes the Frog.fm framework to create interactive Frames that can be deployed to Vercel or any edge runtime.

## Features
* **Frog.fm Core:** Uses the latest Frog framework for high-performance frames.
* **Neynar Integration:** Ready for Neynar API hooks to validate users.
* **Zero Subfolders:** Flat structure for immediate clarity.
* **Edge Ready:** Optimized for Vercel Edge functions.

## Setup
1. Install dependencies: `npm install`
2. Set up environment variables for your Hub or API keys.
3. Run locally: `npm run dev`
